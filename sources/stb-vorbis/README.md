# stb-vorbis

Ogg Vorbis audio decoder (single-file C implementation)

- **Version**: 1.22.0
- **License**: MIT / Public Domain
- **Type**: Single-file C library (requires compilation)
- **Source**: [nothings/stb](https://github.com/nothings/stb)

## Usage

Add to your `ezmk.toml`:
```toml
[depends]
lib = ["stb-vorbis"]
```
