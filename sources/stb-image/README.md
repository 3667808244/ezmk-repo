# stb-image

Image loading/decoding (PNG, JPEG, BMP, TGA, HDR, PSD, GIF, etc.)

- **Version**: 2.30.0
- **License**: MIT / Public Domain
- **Type**: Header-only (single-file C library)
- **Source**: [nothings/stb](https://github.com/nothings/stb)

## Usage

Add to your `ezmk.toml`:
```toml
[depends]
lib = ["stb-image"]
```
